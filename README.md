# Dict
---
翻译——基于百度API的翻译脚本, mac下的翻译工具

---
### Install

	wget https://raw.githubusercontent.com/m2shad0w/Dict/master/install.sh
	sh install.sh
	
### Usage
	英文 －> 中文
	$ dict dict
	###################################
	#  dict 字典
	###################################
	
	$ dict 字典 中国
	$ 多关键字翻译
	###################################
	#  字典 中国 Dictionary China
	###################################
